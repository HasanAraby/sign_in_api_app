# sign_in_api_app

using this app user can:
  - sign up
  - sign in
  - log out
  - read and update his data
  - delete user
  - data is { name , email , phone number , location , personal picture}

used packages:
  - dio
  - get
  - shared_preferences
  - jwt_decoder
  - image_picker
  - dartz

design pattern and state management:
     - mvc and getx.
